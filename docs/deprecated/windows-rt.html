гҖҢWindows RTгҖҚгҒҜгҖҒжҗәеёҜйӣ»и©ұгҒ§еӨҡгҒҸз”ЁгҒ„гӮүгӮҢгҒҰгҒ„гӮӢARMгғҒгғғгғ—гӮ’жҗӯијүгҒ—гҒҹгғҮгғҗгӮӨгӮ№дёҠгҒ§зЁјеғҚгҒҷгӮӢMicrosoftгҒ®OSгҒ§гҒӮгӮҠгҖҒеӨ§гҒҚгҒҸгғҮгӮ¶гӮӨгғігҒҢеӨүгӮҸгҒЈгҒҹгҖҢWindows 8гҖҚгҒЁгҒЁгӮӮгҒ«2012е№ҙеҫҢеҚҠгҒ«еёӮе ҙгҒ«жҠ•е…ҘгҒ•гӮҢгҒҹгҖӮWindows RTгҒҜзҷ»е ҙеҫҢгҖҒиӢҰжҲҰгӮ’еј·гҒ„гӮүгӮҢгҖҒгғҰгғјгӮ¶гғјгӮ’еј•гҒҚгҒӨгҒ‘гӮӢгҒ“гҒЁгҒҢгҒ§гҒҚгҒӘгҒӢгҒЈгҒҹгҖӮгҒқгҒ®зҗҶз”ұгҒ«гҒҜгҖҒгҖҢOutlookгҖҚгҒ®гӮҲгҒҶгҒӘеҫ“жқҘгҒ®Windowsгғ—гғӯгӮ°гғ©гғ гҒҢеҗҢOSдёҠгҒ§еӢ•дҪңгҒ—гҒӘгҒӢгҒЈгҒҹгҒӣгҒ„гӮӮгҒӮгӮӢгҖӮгҒҫгҒҹMicrosoftгҒҜгҖҒгӮҲгӮҠе„ӘгӮҢгҒҹиЈҪе“ҒгӮ’дҪңгӮҠдёҠгҒ’гӮӢгҒҹгӮҒгҒ«Windows RTгғҮгғҗгӮӨгӮ№гҒ®й–Ӣзҷәгғ—гғӯгӮ»гӮ№гӮ’еҺігҒ—гҒҸзөұеҲ¶гҒ—гҖҒеҚ”жҘӯгҒҷгӮӢгғҒгғғгғ—гғЎгғјгӮ«гғјгҒ®ж•°гӮ’зөһгӮҠиҫјгӮ“гҒ гҖӮгҒ—гҒӢгҒ—гҒқгӮҢгҒ«гӮҲгҒЈгҒҰгҖҒWindows RTгӮ’жҗӯијүгҒ—гҒҹиЈҪе“ҒгҒҜеҗҢзӨҫгҒ®гҖҢSurfaceгҖҚд»ҘеӨ–гҒ«гҒ»гҒЁгӮ“гҒ©зҷ»е ҙгҒ—гҒӘгҒ„гҒЁгҒ„гҒҶзөҗжһңгҒ«гҒӘгҒЈгҒҹгҖӮ

з§ҒгҒҜжүӢгӮ’еҮәгҒ—гҒҫгҒӣгӮ“гҒ§гҒ—гҒҹгҒҢгҖҒгҒ“гҒ®й ғгҒҜ Windows XP / Windows 7 гҒҢе…ЁзӣӣгҒ®жҷӮжңҹгҒ§гҒ—гҒҹгҖӮ
