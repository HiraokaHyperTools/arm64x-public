Copilot+ PC の要件とは

改めて確認したいのが Copilot+ PC の要件とは、どのようなものなのでしょうか。

こちらのページによると: Copilot+ PC 開発者ガイド | Microsoft Learn

Copilot+ PC は Windows 11 ハードウェアの新しいクラスで、1 秒あたり 40 兆回を超える演算 (TOPS) を実行できる、リアルタイム翻訳や画像生成などの AI 集中型プロセス用の特殊なコンピューター チップである、高性能ニューラル処理ユニット (NPU) を搭載しています。

このような説明があり、別段 Arm CPU が必須であるという風な記載は見当たりませんでした。 そうだったのか… であれば「Qualcomm の Arm CPU でなくとも良いよね」となりませんか。
