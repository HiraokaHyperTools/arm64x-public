  • CyberLink サポート センター

クアルコム® の CPU を搭載した ARM 版 Windows 11 プラットフォームに対応する CyberLink のソフトウェア バージョンは

以下の CyberLink ソフトウェア バージョンは、クアルコム® 社製の CPUを搭載した ARM 版 Windows 11 プラットフォームに対応しています。 ただし、これらのプラットフォームでは、使用中の Windows OS バージョンや CPU アーキテクチャの違いにより、 (Windows x86/x64 PC と比較して)機能が制限される場合があります。 また、ARM 版 Windows PC で使用する際には、機能に違いが生じる可能性があることにご注意ください。
